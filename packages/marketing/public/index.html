ClientPad

Open-source WhatsApp-first CRM infrastructure for service businesses and developers building client workflows.

MIT Licensed

Turn WhatsApp into your business backend

ClientPad converts everyday WhatsApp conversations into structured CRM activity — leads, bookings, payments, follow-ups, reviews, and client pipelines — without locking you into a hosted platform.

What ClientPad handles

Lead Capture

WhatsApp messages automatically create leads with contact info, source, and intent detection.

Service Bookings

Clients can request and confirm bookings through WhatsApp conversations. Pipeline stages track every job from lead to completed.

Provider-Based Payments

Payment links and confirmations through supported providers. Track paid status directly in the client pipeline.

Smart Follow-Ups

Automated follow-up prompts and review requests keep clients engaged after service completion.

Client Pipeline

Visual pipeline from New Lead → Quoted → Booked → In Progress → Completed → Paid → Review Requested.

Team Inbox

Shared WhatsApp inbox with assignment, AI-suggested replies, and owner-approval workflows.

Built for developers

REST API

Versioned public API at /api/public/v1. Manage leads, clients, and usage with bearer token auth.

TypeScript SDK

@clientpad/sdk — typed client with auto-complete. Works in Node.js, Deno, Bun, and edge runtimes.

CLI Tool

clientpad init, migrate, api-key create, and WhatsApp project scaffolding in one command.

Bring Your Own DB

PostgreSQL-backed. No vendor lock-in. Run migrations locally and deploy anywhere.

Optional Cloud

ClientPad Cloud

Hosted PostgreSQL, managed API endpoint, usage analytics, rate limiting, and a developer dashboard. Self-hosted API keys remain free and unlimited.
