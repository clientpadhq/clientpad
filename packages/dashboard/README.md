# ClientPad Dashboard

Developer web interface for ClientPad Cloud.

```bash
pnpm add @abdulmuiz44/clientpad-dashboard
```

For local development:

```bash
pnpm --filter @abdulmuiz44/clientpad-dashboard dev
```

The dashboard connects to a deployed `@abdulmuiz44/clientpad-cloud` control plane and uses `CLIENTPAD_CLOUD_ADMIN_TOKEN` as the operator login token.

## Screens

- Login
- Overview
- Projects
- API keys
- Usage activity
- Plans and billing
- Docs quickstart
- Settings
